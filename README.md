### HTML5 Music Player (alpha)

#### [Check the live version](http://portfolio.mrcdk.com/html5-music-player/)

##### Only tested with Firefox

To compile you will need the latest development versions of almost all [thx-lib](http://thx-lib.org/) libraries, [Doom](https://github.com/fponticelli/doom), [Dots](https://github.com/fponticelli/dots) and [PushState](https://github.com/jasononeil/hxpushstate).

#### Sources supported
- [VIP Aersia](http://vip.aersia.net/): VIP, Mellow, Source, Exiled, WAP and CPP
- [JetSetRadio](jetsetradio.live/)

#### Known issues
- The loading progress bar is disabled because it doesn't work correctly.
- The player card can't be hidden.
